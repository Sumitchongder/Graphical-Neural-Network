# Code of Conduct

## Our Commitment

We are committed to fostering a professional, inclusive, and respectful environment for everyone involved in this project.

This repository focuses on **Graph Neural Networks (GNNs), Large Language Models (LLMs), hybrid AI systems, and research-grade experimentation**. As such, we value scientific rigor, ethical conduct, and open collaboration.

---

## Expected Behavior

All participants are expected to:

- Communicate respectfully and professionally
- Encourage inclusive and constructive discussions
- Respect differing viewpoints, backgrounds, and expertise
- Provide evidence-based feedback and critique
- Acknowledge prior work, limitations, and assumptions honestly
- Uphold reproducibility and research integrity

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or offensive language
- Personal attacks, insults, or hostile communication
- Plagiarism or misrepresentation of results
- Sharing private or sensitive information without consent
- Any behavior that undermines trust, collaboration, or scientific integrity

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- Code reviews and comments
- External communications representing the project

---

## Enforcement

Project maintainers may take appropriate action in response to violations, including:

- Issuing warnings
- Removing comments or contributions
- Temporarily or permanently restricting participation
